# 🐾 Animal Shelter Management System (Static Demo)

## 📌 Overview

This project is a **frontend-based Animal Shelter Management System** designed to simulate a complete web application without using any backend or database.

Originally built using **PHP + Oracle**, the system has been fully converted into a **static HTML, CSS, and JavaScript project** for easy deployment and demonstration.

👉 The system demonstrates both:

* 👩‍💻 **Admin Panel**
* 👤 **User Interface**

All data is **hardcoded (dummy data)** to mimic real-world functionality.
